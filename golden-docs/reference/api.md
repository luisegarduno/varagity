# HTTP API

The FastAPI service (spec_v2 §4): the app factory, the wire contract, SSE
framing, dependencies, and the route modules. The HTTP-level contract
(paths, bodies, event protocol) is documented as-built in the OpenAPI
schema served at `/openapi.json`; these are the Python internals.
