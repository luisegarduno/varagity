# Chunking

The strategy registry and its five members: `recursive_character` (the v1
default) plus the v2 additions `token_based`, `markdown_aware`,
`docling_hybrid`, and `semantic`. Mind that `CHUNK_SIZE`'s unit is
per-strategy (characters vs tokens — see [Configuration](config.md)).
